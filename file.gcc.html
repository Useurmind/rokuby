GCC Project Compilation

Creating your software project with the gcc means to setup a build process that handles the coordination of compiling single source files and linking the create object files together with perhaps other premade libraries or binary code from other projects.

Rokuby provides a way to do this in a similar fashion to creating visual studio projects (here visual studio handles the coordination for the compilation and linking process). Because we want to make this build process similar to the visual studio process we have similar and even some shared logical constructs (remember the general project build process).
